# Bio-Inspired Robot Snake

This project is a bio-inspired modular robot snake capable of lateral undulation locomotion, UART-based distributed control, and real-time object detection using YOLO on a Raspberry Pi 5.

The project combines:
- Snake locomotion
- UART serial communication
- Embedded systems
- Real-time computer vision
- Distance estimation
- Wireless monitoring

---

# Hardware Overview

## Snake Body

The robot snake is composed of modular links with:

- High torque 64kg servo motors
- Raspberry Pi Pico microcontroller
- Custom voltage regulation circuitry
- Custom housing structure

The snake currently uses lateral undulation to move forward.

---

# Power System

The robot is powered using a 12V battery.

A custom voltage regulator circuit is used to generate:
- 8V rail for servo motors
- 5V rail for Raspberry Pi Pico

The regulator system uses:
- 8V fixed regulator
- 5V fixed regulator
- Supporting capacitors and transistor circuitry

---

# Snake Head

The head module contains:

- Raspberry Pi 5
- Hiwonder AI camera
- Buck converter
- Battery system
- Two small servos (currently unused)

The Raspberry Pi 5 handles:
- Image processing
- YOLO object detection
- Wireless streaming
- High level processing

# Software Structure

## robot_snake_host/

Contains the majority of:
- Locomotion code
- UART communication
- Servo control
- Snake motion generation

### Main File

## go_forward.py

This file generates the lateral undulation motion.

Users can modify:
- Number of servos
- Wave amplitude
- Wave speed
- Center position
- Phase shift

The snake is currently hardcoded to move straight forward until stopped.

The locomotion equation is based on a sinusoidal lateral undulation model.

---

# camera_code/

Contains the majority of:
- Computer vision
- Object detection
- Distance estimation
- Image processing algorithms

---

## offset.py

Uses:
- YOLO object detection
- Pinhole camera principle

Outputs:
- Approximate object distance
- Left/right offset from image center

This file is used for basic obstacle awareness and visual monitoring.

---

## 2frames.py

Experimental image processing algorithm that estimates depth using:
- Two consecutive image frames
- Motion-based depth approximation

This approach attempts depth estimation without requiring stereo cameras.

---

## combined.py

Combines:
- Snake locomotion
- UART communication
- Object detection
- Image processing

The processed camera output is streamed to a local host page for wireless monitoring.

---

# Wireless Access

The snake can be controlled wirelessly by SSH into the Raspberry Pi.

Example:

```bash
ssh pi@RASPBERRY_PI_IP
```

The camera stream can also be viewed wirelessly through the local host stream generated by the vision system.

# Dependencies

This project requires:

- Python 3
- OpenCV
- Ultralytics YOLO
- NumPy
- PySerial

YOLO must be installed separately.

Example installation:

```bash
pip install ultralytics
pip install opencv-python
pip install pyserial
```

Some Raspberry Pi camera dependencies may also be required depending on the camera setup.
